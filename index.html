PEC 2 - Desarrollo web en entorno cliente

Actividad 1

En esta práctica, construirás una aplicación web que obtenga y muestre una lista de usuarios desde una API pública.

Objetivos

  • Obtener una lista de usuarios desde "https://jsonplaceholder.typicode.com/users" utilizando fetch().
  • Mostrar los nombres, correos electrónicos y ciudades de los usuarios en una tabla.
  • Al hacer clic en "Ver detalles", obtener información adicional del usuario seleccionado y mostrarla en pantalla.
  • Manejar posibles errores en las solicitudes de red. Ver mas...

Actividad 2

En esta práctica, crearás una aplicación que permita buscar imágenes en la API de Unsplash.

Objetivos

  • Capturar un término de búsqueda ingresado por el usuario.
  • Capturar la cantidad de imágenes que se quieren mostrar.
  • Hacer una solicitud a "https://api.unsplash.com/search/photos" usando fetch().
  • Mostrar las imágenes obtenidas en la página.
  • Manejar posibles errores en la solicitud. Ver mas...
